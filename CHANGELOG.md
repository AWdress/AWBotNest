# 更新日志

本项目所有重要变更都记录在此。版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## v1.0.0

首个正式版本。

- 平台内核 + 单文件插件：一个功能 = 一个 `.py` 文件，上传即用，真热插拔（启用/停用不重启进程）。
- 插件市场：内置官方仓库 `AWdress/AWBotNest-Plugins`，可配置多个 GitHub 仓库，浏览并一键安装插件（只下载不自动启用）。
- 多账号：网页登录 Telegram 账号，可同时管理多个；插件支持按账号选择生效范围，不同号可开不同插件。
- Web 控制台：插件管理、账号管理、运行日志、系统状态、系统设置，全在深色界面中操作。
- 通知中心：插件通过 `ctx.notify` 提交，平台统一分类、标注账号、推送给平台主人。
- 用户名+密码登录（PBKDF2 哈希 + 无状态 HMAC 令牌）。
